팰린드롬으로 만들기

문제

길이 $N$의 알파벳 대문자로만 이루어진 문자열 $S$가 주어진다. 당신은 아래의 연산을 $0$회 이상 사용할 수 있다.

  • $S$의 문자 하나를 고르고, 다른 알파벳 대문자로 바꾼다.

$S$를 팰린드롬으로 만들기 위해 사용해야 하는 연산의 최소 횟수, 그리고 연산을 사용한 이후의 문자열을 알고 싶다.

입력

첫째 줄에 $N$이 주어진다. ($1 \le N \le 300$)

둘째 줄에 $S$가 주어진다.

출력

첫째 줄에 $S$를 팰린드롬으로 바꾸기 위해 사용해야 하는 연산의 최소 횟수를 출력한다.

둘째 줄에 해당하는 연산 횟수를 사용한 이후 팰린드롬이 된 문자열을 출력하되, 가능한 답이 여럿인 경우 사전순으로 가장 앞서는 문자열을 출력한다.

ABBA는 이미 팰린드롬이므로, 연산을 사용할 필요가 없다.

ABCC에서 $3$번째 문자를 C → B로 바꾸고, $4$번째 문자를 C → A로 바꾸면, 팰린드롬 문자열인 ABBA가 된다. $2$회보다 더 적은 연산을 사용하여 팰린드롬을 만들 수 있는 방법은 없다.
