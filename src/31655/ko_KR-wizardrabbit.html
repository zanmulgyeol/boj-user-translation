골때리는 날짜 포맷

문제

날짜 포맷을 처리하는 방식은 꽤나 다루는 것이 골치아프다. 왜냐하면 국가마다 이러한 포맷이 다르기 때문이다.

예를 들어, 미국에서는 MM/DD/YYYY과 같은 포맷을 사용하지만, 유럽에서는 DD/MM/YYYY과 같은 포맷을 사용한다.

미국, 유럽 중 하나에서 사용하는 날짜 포맷이 주어지면, 해당 포맷이 어느 종류의 포맷인지를 판별해 보자. 단, 한 달은 항상 $31$일이다.

입력

첫째 줄에 $AA$/$BB$/$CCCC$와 같은 형식으로 입력이 주어진다. ($1 \le AA \le 31$; $1 \le BB \le 31$; $0 \le CCCC \le 9\,999$)

미국, 유럽 두 포맷 모두가 아닌 입력은 주어지지 않는다.

출력

만약 입력으로 주어진 포맷이 두 포맷 모두가 가능한 경우라면 either를 출력한다.

그렇지 않다면, 입력으로 들어온 포맷이 미국에서 사용되는 포맷일 경우 US, 아니라면 EU를 출력한다.
