k-정렬

문제

컴퓨터공학과 새내기인 당신은 당신의 이름을 딴 알고리즘을 만드는 것이 꿈입니다. 정렬 알고리즘을 배운 당신은 새로운 정렬 알고리즘을 개발했습니다. 당신이 개발한 알고리즘은 $k$로 나눈 나머지가 같은 두 수의 위치를 교환하여 정렬하는 알고리즘입니다.

정수로 이루어진 배열이 주어질 때 당신이 개발한 알고리즘으로 오름차순 정렬이 가능한지 판단해 봅시다.

입력

첫 번째 줄에 두 개의 정수 $n$ ($1 \le n \le 1000$) 및 $k$ ($1 \le k \le 10^9$) 가 공백으로 구분되어 주어집니다. $n$은 배열의 길이, $k$는 본문에 언급된 값입니다.

두 번째 줄에 $n$개의 정수 $a_i$ ($1 \le a_i \le 10^9$)가 공백으로 구분되어 주어집니다.

출력

당신이 개발한 알고리즘으로 주어진 배열을 오름차순으로 정렬할 수 있다면 YES, 아니면 NO를 출력합니다.
