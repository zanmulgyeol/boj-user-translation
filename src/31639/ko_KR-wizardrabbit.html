자릿수의 개수의 합

문제

길이 $N$의 정수 수열 $A_1, A_2, \cdots, A_N$와 $B_1, B_2, \cdots, B_N$가 주어진다. $\mathrm{digit}(x)$를 $x$의 자릿수의 개수라 할 때, 아래의 값을 구하여라.

$$\sum_{i = 1}^{N} \sum_{j = 1}^{N} \mathrm{digit}(A_i + B_j)$$

입력

입력은 아래와 같이 주어진다.

$N$

$A_1 \; A_2 \; \cdots A_N$

$B_1 \; B_2 \; \cdots B_N$

출력

문제의 정답을 출력한다.

제한

  • $1 \le N \le 150\,000$
  • $1 \le A_i \le 999\,999\,999$ ($1 \le i \le N$)
  • $1 \le B_j \le 999\,999\,999$ ($1 \le j \le N$)
  • 입력으로 주어지는 모든 수들은 정수

서브태스크

번호 배점 제한

$N = 1$

추가 제약 조건 없음

보다 명확한 설명을 위해 $A$와 $B$를 이용하여 만들 수 있는 수들을 행렬에 모두 적어보자. 행렬의 $i$행 $j$열에 적힌 수는 $A_i + B_j$이다.

만들 수 있는 $9$개의 수들은 $117$, $99$, $118$, $99$, $81$, $100$, $27$, $9$, $28$이 되며, 각각의 자릿수는 $3$, $2$, $3$, $2$, $2$, $3$, $2$, $1$, $2$이다. 자릿수를 모두 더하면 $20$이 되므로, 답은 $20$이다.

이 예제는 서브태스크 $2$, $3$, $8$의 조건을 만족한다.

$16$개의 정수의 자릿수의 합은 $46$이므로, 답은 $46$이다.
